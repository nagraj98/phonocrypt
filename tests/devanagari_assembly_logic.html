Token types
कConsonant — has inherent अ, needs halant to suppress it
आVowel standalone — used at word start or after another vowel
ाVowel matra — diacritic form, attached to preceding consonant
्Halant — suppresses inherent अ, joins two consonants into a cluster
∅Inherent vowel (ə / ʌ) — write nothing after a consonant, write अ elsewhere
ˈStrip — stress markers, not sounds
The 3 assembly rules
Rule 1 — Consonant after consonant → insert halant (्) between them
Two consonants in a row means no vowel between them. Halant tells Devanagari to suppress the inherent अ of the first consonant and merge the two into a conjunct.
k + w → क + ् + व = क्व
Rule 2 — Vowel after consonant → use matra form (not standalone)
A vowel following a consonant becomes a diacritic mark on that consonant, not a separate letter.
k + ɪ → क + ि = कि  (NOT क + इ)
b + ɑː → ब + ा = बा  (NOT ब + आ)
Rule 3 — Schwa (ə / ʌ) after consonant → write nothing
The inherent vowel of every Devanagari consonant IS the schwa. Writing अ explicitly would double it.
ð + ə → द (just द, not दअ)
If schwa appears at word start or after another vowel → write अ explicitly.
